# SaveMyMoney

It is a basic site to save money from your friends.

## Technologies

Technologies used in this application:

- PHP 7.3.24 (Backend).
- Javascript (Frontend).
- MySQL 8.0.23 (Database).
- Google Analytics (Tracking websites).
